# Capstone Project for Startocode

A responsive front-end landing page and product showcase built as part of the Startocode curriculum. This project demonstrates core competencies in semantic HTML5 markup, adaptive layout design via CSS3 (Flexbox/Grid), and client-side interactivity using dynamic JavaScript.

## 📁 Repository Structure

```font
Capstone-for-Startocode/
│
├── index.html          # Main HTML structure with semantic sections
├── style.css           # Modular and responsive presentation layer
├── script.js          # Client-side validation and interactive elements
└── Images/             # Visual assets
    ├── EMS.svg         # Corporate identity / Branding vector
    ├── check.svg       # UI indicator icon
    ├── heroImage.png   # Main feature image banner
    ├── heroImage.svg   # Vector format main banner alternative
    ├── product1.png    # Product variant showcase 1
    ├── product2.png    # Product variant showcase 2
    ├── product3.png    # Product variant showcase 3
    ├── product4.png    # Product variant showcase 4
    ├── product5.png    # Product variant showcase 5
    └── product6.png    # Product variant showcase 6

```

---

## ✨ Features

* **Responsive Vector Hero Banner:** Utilizing a mixed-media strategy (`.png` and high-fidelity `.svg`) to ensure optimal rendering across high-DPI displays.
* **Grid Showcase Gallery:** A native grid array managing a 6-tier product catalogue (`product1.png` through `product6.png`) that gracefully degrades on smaller viewports.
* **State Indicators:** Functional interface messaging integrated using system vectors (`check.svg`).

---

## 🛠️ Technology Stack

* **HTML5:** Semantic architecture (`<header>`, `<main>`, `<section>`, `<footer>`) to promote search engine optimization (SEO) and strict accessibility compliance.
* **CSS3:** Declarative layouts focusing heavily on responsive modern layout primitives without over-reliance on bulky frameworks.
* **JavaScript (ES6+):** Pure, native vanilla script layer handling event delegation and client runtime tasks.

## 🚀 Getting Started

### Prerequisites

A modern, standard-compliant web browser (Chrome, Edge, Safari, Firefox). No backend runtimes or build steps are required.

### Local Installation

1. Clone the repository to your workstation:
```bash

```



```
2. Navigate directly into the root directory:
   ```bash
   cd Capstone-for-Startocode-b229e316267a415ec34cacd2a0f6d4c99dc6857a

```

3. Boot the application file:
* **Option A:** Double-click `index.html` to open it locally.
* **Option B (Recommended):** Serve via a local development server such as VS Code's *Live Server* extension to ensure flawless absolute path mappings and resource fetching.
